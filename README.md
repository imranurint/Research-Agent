# Research  Agent


A small Python application for running a research-oriented agent workflow.

## Project Structure

- `main.py` – entry point for the application
- `model.py` – model-related logic
- `tools.py` – tool definitions and helpers
- `system_prompt.py` – system prompt configuration
- `templates/` – HTML templates used by the app
- `requirements.txt` – Python dependencies

## Setup

1. Create and activate a virtual environment (optional but recommended).
2. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

3. Run the application:

   ```bash
   python main.py
   ```
